# 👟 Nike Shoes Landing Page

Nike Shoes Landing Page is a modern and animated product landing page built using HTML, CSS, and JavaScript.  
It features stylish Nike shoes with floating effects, smooth transitions, and an interactive user interface.

---

## 🚀 Live Demo
🔗 https://ps-nike-landing.netlify.app/

---

## ✨ Features
- 👟 4 different shoe color variants  
- ✨ Smooth floating animations  
- 🎨 Modern and attractive UI design  
- ⚡ Interactive transitions and effects  
- 📱 Fully responsive landing page  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ How It Works
- Users can view different Nike shoe variants  
- Smooth animations create an engaging experience  
- Floating effects enhance the modern UI feel
